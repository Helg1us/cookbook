# Workflow Results

This directory is reserved for future RepoPrompt CE workflow evaluation records.

Use it to track:

- workflow version or commit;
- task/domain tested;
- outcome achieved;
- validation evidence;
- reviewer or Oracle scores;
- costs, iteration counts, and notable tradeoffs;
- comparison against earlier workflow versions or variants.

Keep result records grounded in concrete evidence: workflow version, task context, commands or reviews run, and scoring rationale.
